# 🤖 Codeforces Companion Discord Bot

A powerful, feature-rich Discord bot tailored for **competitive programmers** on Codeforces, CodeChef, and LeetCode.

---

## ✨ Features Overview

| Category             | Command(s) / Feature                            | Description                                                                 |
|----------------------|--------------------------------------------------|-----------------------------------------------------------------------------|
| ✅ Verification       | `/verify`                                       | Verifies user via Codeforces and assigns rating role                        |
| ⚔️ Duels              | `/duel`, `/duelleaderboard`, `/myduelpoints`   | 1v1 duels with live problems, tracked leaderboard & scoring                 |
| 🧠 Practice Tools     | `/recommendcf`, `/trainingplan`, `/dailyproblem`| Suggests personalized practice problems or daily challenges                 |
| 📊 Compare Users      | `/comparecf`, `/comparediscord`, `/comparemulti`| Visual graph of rating history across one or multiple users                 |
| 🔥 CF Heatmap         | `/cfheatmap`                                    | GitHub-style heatmap showing daily accepted submissions                     |
| 📅 Contest Reminders  | `/setreminderchannel`, `/disablecontestsite`   | Reminders 5 hours before CF/CC/LC contests (with customizable message)      |
| 🧩 POTD System        | `/setpotd1week`, `/viewpotd`, `/editpotd`      | Auto-post daily problems in designated channels with roles & time control   |
| 🏆 Solve Hunt         | `/solvehunt`                                    | Weekly 5-problem challenge; fastest solvers win a special role              |
| 🫶 Thank System       | `/thank`                                        | Sends a thank-you card & tracks karma                                      |
| 💬 Suggestions        | `/suggestion`, `/setmodchannel`                | Lets users send feedback/suggestions to mod channel                         |
